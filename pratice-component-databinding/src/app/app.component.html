Tugas

  1. Buatlah tiga component: Game Kontrol, Genap dan Ganjil
  2. Component Game Kontrol memiliki button Start dan Pause
  3. Ketika tombol Start game ditekan akan menghasilkan event(dan juga menambah nilai) dan juga menghasilkan emitted second (ref = setInterval())
  4. Event akan diaktifkan dari luar component
  5. Ketika tombol Stop diklik, event get emmited dipanggil (clearInterval(ref))
  6. Di component Ganjil, akan meng-emminted nilai ganjil, begitupun component Genap
  7. Simply output nomor Ganjil - Genap
  8. Berikan style pada masing-masing keduanya
